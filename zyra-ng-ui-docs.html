Installation

Get up and running

Zyra NG UI requires Angular 19+ with standalone components enabled.

1
Install the package
Terminal
npm install zyra-ng-ui
2
Add styles to angular.json
angular.json
"styles": [
  "node_modules/zyra-ng-ui/styles/index.scss",
  "src/styles.scss"
]
3
Provide Zyra in app.config.ts
app.config.ts
import { provideZyra } from 'zyra-ng-ui';

export const appConfig = {
  providers: [
    provideZyra({ theme: 'dark', respectSystemTheme: true })
  ]
};
4
Import and use any component
any.component.ts
import { ZyraButtonComponent } from 'zyra-ng-ui';

@Component({
  standalone: true,
  imports: [ZyraButtonComponent],
  template: `<zyr-button variant="primary">Click me</zyr-button>`
})
Signal Architecture

How signals power Zyra

Every component uses signal(), computed(), and input() — Angular 19's modern reactive primitives.

💡
Why signals? Fine-grained reactivity means only DOM parts that depend on a changed signal re-render — significantly more performant than zone-based change detection.
button.component.ts
@Component({ selector: 'zyr-button', standalone: true, changeDetection: ChangeDetectionStrategy.OnPush })
export class ZyraButtonComponent {
  variant  = input<string>('primary');
  size     = input<string>('md');
  loading  = input<boolean>(false);
  disabled = input<boolean>(false);
  // Computed — only re-runs when inputs change
  classes  = computed(() => `zyr-btn zyr-btn--${this.variant()}`);
  clicked  = output<MouseEvent>();
}
