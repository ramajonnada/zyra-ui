Zyra UI ships zero runtime peer dependencies — no additional CSS frameworks, icon libraries, or animation packages required. Just install zyra-ng-ui and you're set. All components are standalone by design, but you can still import them into an NgModule via the imports array of any module. The standalone architecture does not prevent module-based usage.
